Enables the SyncFu group buying widget on all pages.

== Description ==

This plugin adds the SyncFu widget to your wordpress blog.
It requires an active SyncFu.com account.

For more information visit: [SyncFu](http://www.syncfu.com)

For any questions or issues, please visit: [SyncFu Support](http://syncfu.com/pages/contact.html)

== Installation ==

1. Upload `syncfu` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the settings (Admin > Settings > SyncFu), enter the 40 character API key provided in the SetUp or Account Settings tab of SyncFu merchant interface
4. While logged in at www.syncfu.com, create offers and afterwards paste the given button codes where desired in your wordpress.
